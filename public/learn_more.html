Stackdriver Logging is a fully managed service that performs at scale. It can ingest application and system log data from thousands of VMs and, even better, analyze all that log data in real time. In this fundamental-level Quest, you learn how to store, search, analyze, monitor, and alert on log data and events from Google Cloud Platform. The labs in the Quest give you hands-on practice using Stackdriver Logging to maximize your learning experience and provide insight on how you can utilize Stackdriver Logging to your own GCP Environments.
Fundamentals of Stackdriver Logging
Using BigQuery and Stackdriver to Analyze BigQuery Usage
Logging with Stackdriver on Kubernetes Engine
Monitoring and Logging for Cloud Functions
Creating and Alerting on Logs-based Metrics
